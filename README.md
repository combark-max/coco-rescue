# 코코 구출 작전

휴대폰 브라우저에서도 플레이할 수 있는 탑다운 구조 게임 프로토타입입니다.

## 실행

파일을 바로 열어도 동작합니다.

```powershell
cd C:\Users\USER\Documents\Codex\2026-06-02\github\work\coco-rescue
python -m http.server 8765 --bind 127.0.0.1
```

그 다음 브라우저에서 `http://127.0.0.1:8765`로 접속합니다.

## 조작

- 휴대폰: 왼쪽 원형 스틱으로 이동과 조준, 오른쪽 빨간 버튼으로 발사
- PC: WASD 또는 방향키로 이동, Space로 발사

## 현재 구현

- 하늘에서 내려보는 탑다운 시점
- 귀여운 2D 만화풍 캐릭터와 길 배경
- 상단 설정에서 캐릭터 4종 선택: 준영, 준성, 아빠, 엄마
- 캐릭터별 최고 점수 브라우저 저장
- 1스테이지 클리어 후 2스테이지로 진행
- 2스테이지 바다 배경
- 2스테이지는 기존보다 두 배 긴 거리와 더 많은 악당 등장
- 2스테이지 전용 새 악당 등장
- 2스테이지 마지막에는 중간보스를 쓰러뜨려야 코코 구출 가능
- 게임당 5번 캐릭터 부활
- 자동 전진
- 상단 설정에서 게임 중 무기 선택: 일반총, 산탄총, 기관총, 권총, 수류탄
- 설정을 열면 게임 일시 중단, 닫으면 계속 진행
- 무기별 발사 횟수 제한
- 무기 충전 아이템: 획득하면 모든 무기 탄약 회복
- 회복 하트 아이템: 획득하면 체력 회복
- 가까운 아이템이 살짝 끌려오는 보정
- 적 추적과 피격
- 끝 지점의 코코 구출 승리 조건
- 모바일 화면용 HUD와 터치 버튼

## GitHub Pages 배포

1. 이 폴더의 `index.html`, `styles.css`, `game.js`, `README.md`를 GitHub 저장소 루트에 올립니다.
2. GitHub 저장소에서 `Settings` → `Pages`로 이동합니다.
3. `Build and deployment`의 Source를 `Deploy from a branch`로 설정합니다.
4. Branch를 `main`, 폴더를 `/root`로 선택하고 저장합니다.
5. 잠시 뒤 표시되는 Pages 주소에서 게임을 확인합니다.
